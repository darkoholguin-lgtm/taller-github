# Taller Básico de GitHub

Este repositorio es solo para práctica.

## Objetivo
Aprender a:
- Hacer fork
- Editar archivos
- Crear commits
- Abrir Pull Requests

## Instrucciones
1. Haz un fork de este repositorio
2. Edita este README
3. Agrega tu nombre en la sección de participantes
4. Guarda los cambios con un commit
5. Crea un Pull Request
